What is Flexbox?

Flexbox is a way to simplify the way we position elements on a page. It is not designed to be used on the entire page, but on sections. Flex box is useful for positioning items either individually, or grouped. It is also possible to nest flex containers and style them.

There are two important components:

  1. Flex Container
    • An element on the page which contains flex items.
  2. Flex items
    • All direct child elements of flex containers

display: flex

Any element can be a flex container. Using display: flex; will turn the element into a flex container and will remain at block level. This will then change the behaviour of the child elements.

display: inline-flex

If you don't want your element as a block-level element, we can use display: inline-flex; to create a flex container that is inline

justify-content

We can use justify-content to position items from left to right. For example:

.container { display: flex; justify-content: flex-end; }

This will cause all the flex items within the container to shift to the right side of the container. There are 5 values for the justify-content property:

Value Description Visual
flex-start Positions items in order from the left of the parent container, with no space between them.
flex-end Positions items in order, with the last item starting on the right side of the parent container, with no extra space in between, or before
center Positions items in the center of the parent container, with no extra space between, before, or after.
space-around Positions items with equal space before and after. This results in double space.
space-between Positions items with equal space between but no extra space before the first, or the last item.
Note: "no extra space" means the margins and borders will be respected but no more space will be added between.

align-items

Using the align-items property, we can position flex items vertically. There are 5 values for align-items:

Value Description Visual
flex-start Position items at the top of the parent container.
flex-end Positions items at the bottom of the parent-container.
center Positions items in the center of the parent container.
baseline Positions items with the bottom all aligned with each other, regardless of height.
stretch If possible, items will stretch from top of the parent container to the bottom. This is the default value. Any element with a specified height will not be stretched.

flex-grow/flex-shrink

Flex items will shrink proportionally when the flex container is too small. If the parent container is larger than necessary, then the flex items will not stretch by default.

This allows us to specify that items should grow or shrink to fill containers when they are resized.

flex-basis

This property allows us to specify the initial width of a flex-item before it shrinks or grows.

flex Property

The flex property provides a convenient way to specify how an element stretches/shrinks. We are able to declare flex-grow/flex-shrink/flex-basis in one line.

If we only use 2 values in the flex property, it will only set the flex-grow and flex-basis.

flex-wrap

Sometimes, we don't want to shrink the items within the container, but instead would rather have it move onto a new line. There are 3 values we can use:

  1. wrap: Child elements that don't fit will move down a row.
  2. wrap-reverse: Child elements that don't fit will move down a row, but in reverse order.
  3. nowrap: This is the default value, and prevents the elements wrapping.

We declare flex-wrap in the flex container.

align-content

When we use align-items we are aligning the items from top to bottom. When we use align-content we are aligning the rows from top to bottom.

This uses 6 values:

  1. flex-start
  2. flex-end
  3. center
  4. space-between
  5. space-around
  6. stretch

flex-direction

Using flex-direction it is possible to switch the main axis and the cross axis. This property accepts 4 values:

  1. row: Flex items will be ordered horizontally.
  2. row-reverse
  3. column: Flex items will be order vertically.
  4. column-reverse

flex-flow

We use this property when we are declaring flex-wrap and flex-direction in one line.
